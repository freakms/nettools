  OPTION 1: Test Without Building (Quickest)
═══════════════════════════════════════════════════════════════════════════════

Run directly from Python to see the new features immediately:

    python nettools_app.py

This will show the history buttons right away!

═══════════════════════════════════════════════════════════════════════════════
  OPTION 2: Rebuild the .exe (For Distribution)
═══════════════════════════════════════════════════════════════════════════════

Single File Version:
    python build_exe.py

Fast Version (Folder):
    python build_exe_fast.py

Or Simplest:
    build_fast_simple.bat

Then run: dist\NetToolsSuite.exe (or dist\NetToolsSuite\NetToolsSuite.exe)

                                   Clock icon button!

  MAC Input: [AA:BB:CC:DD:EE:FF_______] [⏱]
                                          ↑
